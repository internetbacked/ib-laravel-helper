Use Laravel 5.1 to have some feature. Included Laravel Excel by Maatwebsite, HTML & Form builder by Laravelcollective.

== Description ==

What is in the box?

*   Eloquent Model for WooCommerce - Order, Product
*   Eloquent Model - Post, Users, Comments
*   Helpers
    *   [LaravelCollective - Html & Form](https://laravelcollective.com/docs/5.1/html)
    *   [Laravel - Collection methods](https://laravel.com/docs/5.1/collections#available-methods)

== Installation ==

This section describes how to install the plugin and get it working.
